# PhysioLog 시작하기

물리치료사용 환자 차팅 앱 MVP. 환자 차팅 시간 10~20분 → 5분으로 단축이 목표.

라이브 데모: https://physiolog-five.vercel.app (그냥 둘러보기만 하려면 여기로)

---

## 0. 필수 준비물 (한 번만)

**Node.js 20 이상 설치** — https://nodejs.org/ 에서 LTS 다운로드 → 그냥 다음 다음 다음.

설치 확인 (터미널에서):
```
node -v
```
v20 이상이면 OK.

---

## 1. 실행

### Mac 사용자
1. `physiolog.zip` 압축 풀기 (더블클릭)
2. 풀린 폴더에서 `setup.command` **더블클릭**
   - 만약 "확인되지 않은 개발자" 경고가 뜨면 → 시스템 설정 → 보안 및 개인정보 보호 → "그대로 열기"
   - 또는 터미널에서 폴더 들어간 뒤 `bash setup.sh`

### Windows 사용자
1. `physiolog.zip` 압축 풀기 (우클릭 → 압축 풀기)
2. 풀린 폴더에서 `setup.bat` **더블클릭**

### 어느 쪽이든 첫 실행만 1~2분 걸려요 (패키지 다운로드).

---

## 2. 사용

스크립트가 끝나면 자동으로 브라우저가 http://localhost:3000 으로 열림.
안 열리면 직접 주소창에 입력.

**핸드폰에서 보고 싶으면:**
- 컴퓨터와 같은 와이파이에 폰 연결
- 터미널/콘솔 창에 뜬 `Network: http://192.168.xxx.xxx:3000` 주소를 폰 브라우저에 입력
- Safari/Chrome에서 "홈 화면에 추가" 하면 앱처럼 사용 가능

**종료:** 실행 중인 터미널 창에서 `Ctrl + C`

---

## 3. 데이터 저장 위치

- **로그인 없음**, 모든 데이터는 **사용한 브라우저의 localStorage**에 저장됨
- 같은 컴퓨터·같은 브라우저에서만 보임 (다른 폰/PC와 동기화 X)
- 시크릿 모드에서 입력하면 창 닫을 때 다 사라짐 — 일반 창에서 사용

---

## 4. 화면 구성 (4개)

1. **환자 리스트** `/` — 카드, 이름 검색, 우하단 ➕로 환자 등록
2. **환자 정보** `/patients/[id]` — 3탭: 기본정보 / 치료기록 / 평가기록
3. **치료기록** — 부위 다중선택 → 근육 콤보박스 → 치료방법 다중체크 → 운동치료(선택) → 코멘트
4. **평가기록** — VAS / ROM / MMT / 신체계측 토글, recharts 추이 그래프

핵심 기능: **이전 치료기록 1클릭 복사** (같은 환자 같은 치료 반복할 때)

---

## 5. 안 될 때

| 증상 | 해결 |
|------|------|
| `node: command not found` | Node.js 미설치 → 0번부터 |
| `EACCES` / 권한 거부 (Mac) | 터미널에서 `chmod +x setup.sh` 후 재실행 |
| 포트 3000 사용 중 | 다른 dev 서버 종료, 또는 `next dev -p 3001` |
| 페이지 깨짐 | 브라우저 강력 새로고침 (Mac: ⌘+Shift+R / Win: Ctrl+F5) |
| 그래도 안 됨 | 사용자한테 스크린샷 공유 |
